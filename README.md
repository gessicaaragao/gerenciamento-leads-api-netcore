# gerenciamento-leads-api-netcore
#Siga os seguintes passos para executar projeto

1. Navegue até a pasta da API (../GerenciamentoLeads)

2. Restaure os pacotes
bash 
dotnet restore

3. Compile o projeto
bash 
dotnet build

5. Execute a API
bash 
dotnet run

A API estará disponível em:
http://localhost:5000
